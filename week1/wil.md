git bash에는 4가지 명령어를 알아둬야 한다
꼭 알아야될 명령어
git status - 현재 상태
git add <file> - index.html을 넣으면 된다
git commit -m “<메시지>”
git push - github에 업데이트 하는 명령어

단락<p></p>
<tagname>content</tagname>

<h2></h2>
